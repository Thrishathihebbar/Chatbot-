# 🤖 HELPMATE - Chatbot with Context Memory

## 📌 Overview

EmpathAI is an intelligent chatbot developed using **Natural Language Processing (NLP)** and **Machine Learning** techniques. The chatbot is capable of detecting user emotions from text, remembering previous interactions through context memory, and generating relevant responses. The application features a user-friendly interface built with Streamlit, making conversations interactive and personalized.

---

## 🚀 Features

- 😊 Emotion detection from user text
- 🧠 Context-aware conversation with memory
- 💬 Intelligent response generation
- 👤 User name recognition and memory recall
- 🔍 Intent recognition using NLP
- 📊 TF-IDF vectorization and Cosine Similarity
- 🌐 Interactive GUI using Streamlit
- 💾 JSON-based persistent memory
- ⚡ Fast and lightweight application

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Emotion-Aware-Chatbot.git

cd Emotion-Aware-Chatbot
```

### Create Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate Environment

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open automatically in your browser.
